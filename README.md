# Hotel System Interface

## Purpose

This project provides a modern, user-friendly interface for a hotel management system. It is a refined version of code from a group project, redesigned with a brand new UI to make using the system significantly easier and more intuitive.

## Overview

The Hotel System Interface combines HTML and Java to deliver a functional hotel management application with an improved user experience. The system is designed to handle core hotel operations through an accessible and straightforward interface.

## Technology Stack

- **HTML** (55%) - Frontend interface and page structure
- **Java** (45%) - Backend logic and system functionality

## Features

This interface enables users to interact with hotel management features through:
- An intuitive web-based front-end built with HTML
- Robust backend processing powered by Java
- Streamlined workflows for common hotel operations

## Getting Started

1. Ensure you have Java installed on your system
2. Clone the repository
3. Review the HTML files for the interface structure
4. Compile and run the Java components as needed

## Project Background

This is a modified interface from a group project, redesigned from the ground up to prioritize usability and maintainability. The new UI makes the underlying hotel management code accessible and easy to use.
